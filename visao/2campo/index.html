    2. Setting Up the Field

    1. Start Titans-Vision

    Open the terminal and run the following command:

    After clicking to configure the field, a new window will open showing the image captured by the camera, with four overlaid points ‚Äî one for each corner of the field.

    4. Now, use the W, A, S, D keys to move the point in the image:
    5. W ‚Üí up
    6. A ‚Üí left
    7. S ‚Üí down
    8. D ‚Üí right

    üìù Tip: Adjust the points until each one is exactly positioned at the real four corners of the field as projected by the camera.

    After adjusting all four points:

    • Click the Ok button to save the point positions.

    ‚úÖ The system is now calibrated with the correct corners of the field.
